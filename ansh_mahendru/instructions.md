# Online Gaming Website – Task Instructions

## Project Title
Online Gaming Website for NexaPlay

---

## Context

NexaPlay is a gaming startup that wants to launch a browser-based online gaming platform. The target audience is casual and mid-core gamers aged 16 to 35. The platform will allow users to browse a catalog of games, view game details, register and log in to their accounts, track their scores on a leaderboard, and manage their profile.

You are being hired to design the complete UI for this platform. You will deliver high-fidelity designs for all required pages, a mobile preview for two pages, and a style guide.

---

## Requirements

### Pages to Design

You must design the following seven pages:

1. **Homepage**
   - Navigation bar with logo, links to Catalog, Leaderboard, Login, and Sign Up.
   - Hero banner with a featured game image, a headline, a subheadline, and a "Play Now" button.
   - A "Featured Games" section showing at least 6 game cards in a grid layout.
   - A genre filter row with at least 5 genre tags (e.g., Action, Puzzle, Racing, Adventure, Sports).
   - Footer with links to About, Contact, Privacy Policy, and Terms of Use.

2. **Game Catalog Page**
   - A grid of game cards showing at least 12 games.
   - Each game card must display the game thumbnail, title, genre tag, and star rating out of 5.
   - Filter sidebar or top filter bar with options for Genre, Rating, and Popularity.
   - A search bar at the top of the page.

3. **Game Detail Page**
   - Game title displayed as a heading at the top.
   - A large game preview image or banner below the heading.
   - A description section with at least 3 lines of placeholder text.
   - A "Play Now" button placed directly below the description.
   - A star rating display showing the current game rating.
   - A comments or reviews section at the bottom with at least 2 placeholder comments.

4. **User Registration Page**
   - A form with the following fields: Username, Email, Password, Confirm Password, and Date of Birth.
   - A "Create Account" submit button.
   - A link below the form that reads "Already have an account? Log In".

5. **User Login Page**
   - A form with fields for Email and Password.
   - A "Log In" submit button.
   - A "Forgot Password?" link placed below the password field.
   - A link that reads "Don't have an account? Sign Up".

6. **User Profile and Dashboard Page**
   - Display the user avatar placeholder, username, and join date.
   - A stats section showing Total Games Played, Total Score, and Rank.
   - A "Recent Games" section listing at least 3 games with title, score, and date played.
   - An "Achievements" section showing at least 3 badge placeholders with labels.

7. **Leaderboard Page**
   - A table with the following columns: Rank, Username, Game Title, Score, and Date.
   - The table must show at least 10 rows of placeholder data.
   - A filter dropdown to select the game title for which the leaderboard is displayed.

---

## Technical Specifications

- Design tool: Figma or Adobe XD (editable source file must be delivered).
- Desktop canvas size: 1440px wide.
- Mobile canvas size: 375px wide (required for Homepage and Game Catalog Page only).
- Color theme: Dark background (use hex #0D0D0D or similar dark tone) with at least one vibrant accent color.
- Heading font: A bold sans-serif font (e.g., Rajdhani, Exo 2, or Orbitron).
- Body font: A clean readable font (e.g., Inter, Roboto, or Open Sans).
- All buttons must show a default state and a hover state in the design.
- All interactive cards must show a default state and a hover state in the design.

3. Exported PNG mobile previews for two pages at 375px width, named as follows:
   - homepage_mobile.png

4. A style guide page inside the same source file showing:
   - Primary and accent color swatches with hex codes.

   - Body font name and size used.

---

## Scope Boundaries

The following items are outside the scope of this task:

- Backend development of any kind.
- Database design or integration.
- Actual game embedding or playable game content.
- Payment or subscription system design.
- More than 7 page designs.
- Pages not listed in the Requirements section above.
- SEO or accessibility implementation.

---

## Acceptance Gates

Submission will be accepted only if all of the following are true:

- All 7 page designs are present in the source file.
- All 9 PNG exports are present (7 desktop + 2 mobile) with the correct file names listed above.
- The style guide page is present in the source file.
- The source file opens without errors in Figma or Adobe XD.
- The dark background color is applied across all pages.
- Every page includes a navigation bar matching the one described in the Homepage requirements.
- The "Play Now" button is present on both the Homepage hero section and the Game Detail page.
- The Leaderboard table contains the five required columns: Rank, Username, Game Title, Score, and Date.
